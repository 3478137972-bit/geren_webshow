## 🚀 快速开始

### 开发环境

```bash
# 安装依赖
npm install

# 本地开发
npm run dev

# 构建
npm run build

# 预览构建结果
npm run preview
```

### 部署

项目已配置自动部署到 Vercel：
1. 推送代码到 GitHub
2. Vercel 自动构建和部署
3. 访问分配的域名
