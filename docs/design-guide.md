## 二、配色方案

### 2.1 主色调

| 名称 | 色值 | 用途 |
|------|------|------|
| Black | `#000000` | 主标题、重要文字 |
| White | `#FFFFFF` | 背景、反白文字 |
| Gray 100 | `#F9FAFB` | 浅色背景 |
| Gray 500 | `#6B7280` | 次要文字 |
| Gray 900 | `#111827` | 深色文字 |

### 2.2 强调色（待确认）

**方案 A - 蓝色系** (专业、科技感)
```
Primary: #3B82F6
Hover: #2563EB
Active: #1D4ED8
```

**方案 B - 橙色系** (活力、创意)
```
Primary: #F97316
Hover: #EA580C
Active: #C2410C
```

**推荐**: 方案 A（蓝色系）- 更符合 AI/科技定位

## 三、字体规范

### 3.1 字体家族

```css
/* 中文 */
font-family: 'Noto Sans SC', sans-serif;

/* 英文 */
font-family: 'Inter', sans-serif;
```

### 3.2 字号系统

| 用途 | 字号 | 字重 | 行高 |
|------|------|------|------|
| Hero 标题 | 48px / 3rem | 700 | 1.2 |
| 页面标题 | 32px / 2rem | 600 | 1.3 |
| 章节标题 | 24px / 1.5rem | 600 | 1.4 |
| 正文大 | 18px / 1.125rem | 400 | 1.6 |
| 正文 | 16px / 1rem | 400 | 1.6 |
| 小字 | 14px / 0.875rem | 400 | 1.5 |
| 辅助文字 | 12px / 0.75rem | 400 | 1.4 |

### 3.3 字重使用

| 字重 | 值 | 用途 |
|------|-----|------|
| Regular | 400 | 正文 |
| Medium | 500 | 小标题 |
| Semibold | 600 | 章节标题 |
| Bold | 700 | 主标题、强调 |

## 六、组件规范

### 6.1 导航栏

```
高度：64px
背景：白色 / 透明
Logo：左侧
导航项：居中/右侧
移动端：汉堡菜单
```

### 6.2 按钮

| 类型 | 背景色 | 文字色 | 圆角 | 内边距 |
|------|--------|--------|------|--------|
| 主按钮 | #3B82F6 | #FFFFFF | 8px | 12px 24px |
| 次按钮 | 透明 | #3B82F6 | 8px | 12px 24px |
| 边框按钮 | 透明 | #000000 | 8px | 12px 24px |

### 6.3 作品卡片

```
尺寸：自适应（保持图片比例）
圆角：12px
阴影：轻微阴影 (hover 时加深)
悬停效果：轻微上移 + 阴影加深
```

### 6.4 联系表单

```
输入框高度：48px
圆角：8px
边框：1px solid #E5E7EB
焦点状态：蓝色边框 + 轻微阴影
```

## 七、图片规范

### 7.1 作品图片

| 格式 | WebP (优先) / JPG |
|------|------------------|
| 质量 | 80-90% |
| 尺寸 | 最大 1920px 宽 |
| 压缩 | 使用 imagemin 优化 |

```
格式：PNG (透明背景) / SVG

位置：首页 Hero 区、导航栏左侧
```

## 九、交互动效

### 9.1 过渡动画

```css
/* 标准过渡 */
transition: all 0.3s ease;

/* 快速过渡 */
transition: all 0.2s ease;
```

### 9.2 悬停效果

- 按钮：背景色加深 + 轻微上移 (2px)
- 作品卡片：轻微上移 (4px) + 阴影加深
- 链接：下划线显示 / 颜色变化

### 9.3 页面加载

```
Hero 区：淡入 (fade-in)
作品列表：依次淡入 (stagger fade-in)
滚动动画：元素进入视口时触发
```

---

## 十、无障碍设计

### 10.1 对比度要求
- 文字与背景对比度 ≥ 4.5:1 (WCAG AA)
- 大文字对比度 ≥ 3:1

### 10.2 键盘导航
- 所有交互元素可键盘访问
- 焦点状态清晰可见

### 10.3 图片 Alt 文本
- 所有作品图片必须有描述性 alt 文本
- 装饰性图片使用空 alt
